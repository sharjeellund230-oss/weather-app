One Call API 3.0

  • One Call API 3.0

    All-in-One Weather Data, One Simple Call

    The Business

    Access real-time weather, hourly & daily forecasts, national alerts, and historical data — all through a single, streamlined API request. Whether you’re building apps, dashboards, or climate models, One Call API 3.0 is your all-purpose solution.

    • Current Weather

      Temperature, humidity, wind, pressure & more — updated every minute

    • Historical Weather

      Up to 40 years back (hourly/daily) for any location globally

    • Hourly Forecast

      Up to 48 hours ahead, with precise intervals

    • Weather Alerts

      Government-issued national alerts (e.g. NOAA, EUMETNET)

    • Daily Forecast

      Up to 8 days forecast with sunrise/sunset, min/max, and weather icons

    • Minutely Forecast

      1-minute precision for the next hour (in select locations)
